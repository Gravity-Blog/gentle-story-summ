# Gentle Story Summarizer

## Running the Project

1. Make the script executable and run it:  
   ```sh
   chmod +x /path/to/start.sh && /path/to/start.sh
   ```

2. Open the application in your browser at [http://localhost:3000](http://localhost:3000).
